OMX.broadcom.source

source
20

A conformant image source for creating raw bitmap images. The single output port can be configured with the output size, as well as details of how to generate the image.

PortIndexNotes
20OMX_IndexParamPortDefinition Set this to specify the output format. Colour format should be RGB565, RGB888, ABGR8888, YUV420 packed planar, or YUV422 packed planar. Specify nSliceHeight as 16 or leave as zero (and it will use 16 internally). Alternatively to produce full frames set nSliceHeight to the frame size rounded up to the next multiple of 16. Specify nStride as a multiple of 32, or 64 for ABGR8888, or 16 for RGB888, or leave as zero (and it will assign a suitable value internally).
OMX_IndexParamImagePortFormat Query / set the image port format used. The output port supports non-compressed images of a small number of image formats.
OMX_IndexParamSource Set this to configure how the output image will be created.
OMX_IndexParamSourceSeed Set this to control the seed of any random image created.
OMX_IndexConfigBrcmPortStats The output port can be queried for the number of frames, buffers, and bytes produced.
